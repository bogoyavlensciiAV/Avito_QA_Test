## Задание 1: Скриншот с багами

### P0 - Критический

* P0 - Диапазон цен указан от 1 000 до 30 000 ₽, но в выдаче есть объявления за 40 000 ₽ и за 500 ₽ - фильтр по цене не работает *(выделено красным на скрине)*
* P0 - Выбран пол «Мальчик», но в выдаче есть котёнок-девочка - фильтр по полу не работает *(выделено оранжевым на скрине)*
* P0 - В правом верхнем углу отображается системная ошибка «Ошибка, обновите страницу» - пользователь видит техническую ошибку *(выделено жёлтым на скрине)*

### P1 - Высокий

* P1 - Выбран регион «Москва», но в выдаче есть объявление из Новосибирска - фильтр по региону не работает *(выделено зелёным на скрине)*
* P1 - Отображается радиокнопка «Сначала из Можайска» при выбранном регионе «Москва» - опция сортировки не соответствует выбранному региону *(выделено бирюзовым на скрине)*
* P1 - Счётчик в заголовке «Котята в Москве 19» не совпадает с реальным количеством объявлений (отображается 8) - пользователь вводится в заблуждение *(выделено синим на скрине)*
* P1 - В выдачу попало объявление «Персидский котёнок плюшевый» (игрушка за 1 100 ₽) - категория товара не соответствует разделу живых животных *(выделено фиолетовым на скрине)*

### P2 - Средний

* P2 - Некорректный порядок хлебных крошек: отображается «Главная -> Животные -> Персидская -> Кошки» вместо «Главная -> Животные -> Кошки -> Персидская» *(выделено розовым на скрине)*
* P2 - При наведении на иконку «сердечко» (избранное) отображается текст «Не показывайте мне похожие объявления» - действие контринтуитивно, ожидается «Добавить в избранное» *(выделено чёрным на скрине)*
* P2 - У части объявлений отображается дата публикации («7 дней назад»), у других - нет; логика отображения непоследовательна *(выделено коричневым на скрине)*

### P3 - Низкий

* P3 - В фильтре «Возраст» опция называется «Щенок» вместо «Котёнок» - некорректная терминология для раздела кошек *(выделено серым на скрине)*
* P3 - Опечатка в пункте меню: «Коръера в Авито» вместо «Карьера в Авито» *(выделено цветом хаки на скрине)*

## Задание 2: Тесты API
Автотесты для API сервиса объявлений `https://qa-internship.avito.com`.

## Требования

- Java 17+
- Gradle 9+

## Быстрый старт

Склонируйте репозиторий:
```bash
git clone https://github.com/bogoyavlensciiAV/Avito_QA_Test.git
cd Avito_QA_Test
```

Запустите тесты:
```bash
./gradlew test
```


## Отчёт Allure

Сгенерировать и открыть отчёт в браузере:
```bash
./gradlew allureServe
```

Или сохранить отчёт локально в `build/reports/allure-report/`:
```bash
./gradlew allureReport
```

## Линтер и форматтер

Проект использует [Spotless](https://github.com/diffplug/spotless) с правилами Google Java Format.

Проверить форматирование:
```bash
./gradlew spotlessCheck
```

Автоматически исправить:
```bash
./gradlew spotlessApply
```

Правила зафиксированы в блоке `spotless` файла `build.gradle`.
